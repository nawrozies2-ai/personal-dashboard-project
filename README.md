# Personal Dashboard Project

## Project Overview

This project is a full-stack personal dashboard application built with HTML, CSS, JavaScript, Node.js, and Express.js.

The dashboard allows users to:
- update profile information
- manage tasks
- fetch data from a live Express API
- save tasks to the backend

---

## Features

- Express.js backend
- REST API routes
- Dynamic task management
- Form validation
- Async fetch requests
- Loading indicators
- Error handling
- Responsive design
- Hover animations

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js

---

## Environment Variables

PORT=3000
APP_NAME=Personal Dashboard

---

## How To Run The Project

1. Install dependencies:

npm install

2. Start the server:

node server.js

3. Open index.html in the browser

---

## API Routes

GET /api/items

Returns all tasks/items.

POST /api/items

Adds a new task/item to the backend.

---

## Deployment

This project can be deployed using Render or another cloud platform.
